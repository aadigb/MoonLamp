ESP32 Setup Guide

Your ESP32 polls http://<your-ip>:3000/api/color every 60 seconds.

The response now includes intensity (0.0–1.0) and alert (true/false) so the lamp dims/brightens with market moves and flashes on big swings.

  1. Install Arduino IDE and add ESP32 board support.
  2. Install libraries: Adafruit NeoPixel and ArduinoJson.
  3. Open firmware/moonlamp/moonlamp.ino.
  4. Fill in your WiFi SSID, password, and this server's local IP.
  5. Flash to ESP32 via USB.
